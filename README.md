## Claude Skills

Claude Skills are modular, task-specific packages that teach Claude how to execute particular workflows or patterns. Each skill typically contains instructions, scripts, and templates.

### Development

- [Code Review Assistant](https://github.com/example/code-review-skill) - Automated code review following best practices and style guides.
- [API Documentation Generator](https://github.com/example/api-docs-skill) - Generate comprehensive API documentation from code.
- [Test Case Generator](https://github.com/example/test-gen-skill) - Create unit and integration tests based on code analysis.

### Data Analysis

- [Excel Formula Builder](https://github.com/example/excel-formula-skill) - Generate complex Excel formulas based on natural language descriptions.
- [Data Visualization](https://github.com/example/data-viz-skill) - Create charts and visualizations from datasets.
- [Report Generator](https://github.com/example/report-gen-skill) - Generate data-driven reports with insights and recommendations.

### Content Creation

- [Blog Post Writer](https://github.com/example/blog-writer-skill) - Create SEO-optimized blog posts on various topics.
- [Social Media Manager](https://github.com/example/social-media-skill) - Generate engaging social media content across platforms.
- [Documentation Writer](https://github.com/example/docs-writer-skill) - Create technical documentation following documentation standards.

### Business Operations

- [Meeting Summarizer](https://github.com/example/meeting-summary-skill) - Summarize meeting transcripts with action items.
- [Email Responder](https://github.com/example/email-responder-skill) - Draft professional email responses based on context.
- [Project Planner](https://github.com/example/project-planner-skill) - Create detailed project plans with milestones and tasks.

## MCP Servers

Model Context Protocol (MCP) servers provide standardized interfaces for connecting Claude to external tools, databases, and services.

### File Systems

- [MCP Filesystem](https://github.com/modelcontextprotocol/servers) - Access and manipulate files on local or remote filesystems.
- [Google Drive MCP](https://github.com/example/gdrive-mcp) - Integrate with Google Drive for file operations.
- [Dropbox MCP](https://github.com/example/dropbox-mcp) - Connect to Dropbox for cloud storage access.

### Databases

- [PostgreSQL MCP](https://github.com/example/postgres-mcp) - Query and manage PostgreSQL databases.
- [MySQL MCP](https://github.com/example/mysql-mcp) - Interact with MySQL databases.
- [MongoDB MCP](https://github.com/example/mongodb-mcp) - Work with MongoDB collections and documents.

### APIs and Integrations

- [GitHub MCP](https://github.com/modelcontextprotocol/servers) - Integrate with GitHub for repository management.
- [Slack MCP](https://github.com/example/slack-mcp) - Send messages and interact with Slack workspaces.
- [Linear MCP](https://github.com/example/linear-mcp) - Manage issues and projects in Linear.

### Development Tools

- [Git MCP](https://github.com/example/git-mcp) - Execute git commands and manage repositories.
- [Docker MCP](https://github.com/example/docker-mcp) - Manage Docker containers and images.
- [Kubernetes MCP](https://github.com/example/k8s-mcp) - Interact with Kubernetes clusters.

## Contributing

Contributions are welcome! Please read the [contribution guidelines](CONTRIBUTING.md) first.

We're looking for:
- New Claude Skills (with SKILL.md files)
- MCP Server implementations
- Tutorials and guides
- Tools and libraries
- Bug fixes and improvements

Please ensure your pull request adheres to the following guidelines:
- Search previous suggestions before making a new one
- Make an individual pull request for each suggestion
- Use the following format: `[Name](link) - Description.`
- New categories or improvements to the existing categorization are welcome
- Keep descriptions short and simple, but descriptive
- End all descriptions with a period
- Check your spelling and grammar
- Make sure your text editor is set to remove trailing whitespace

Thank you for your suggestions!
